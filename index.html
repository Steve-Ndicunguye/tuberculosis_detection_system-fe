Tuberculosis Detection System

Detect Tuberculosis by uploading a chest x-ray image

How the App Works Use the App

Attention!

This is a prototype. If you are really suffering from tuberculosis I advice acquiring serious medical assistance.

Description:

This web app uses computer vision to detect tuberculosis (TB) on chest x-rays. Today there's still no quick test for TB like there is for Malaria and HIV. This is a step towards creating a test that can quickly identify whether or not a person is infected.

Image format:

The app accepts chest x-ray images and images in jpg or png format.

Limitations:

The Ai brain (model) that powers the app was trained using only 800 x-ray images. This is a very small dataset by deep learning standards. Therefore, even though the scores during testing are reasonably good, I'm not sure how well this model will perform under real-world conditions. Also, the prediction accuracy could be affected by factors like x-ray machine type, varying image lighting conditions, damaged x-ray images (e.g. pen marks) and other factors.
